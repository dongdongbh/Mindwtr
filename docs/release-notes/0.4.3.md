# Mindwtr 0.4.3

## Highlights
- Desktop attachments: the **Add link** field now accepts local file paths (e.g., `/home/user/file.pdf`, `C:\Users\you\file.txt`, `file://...`).
- Diagnostics logging: clearer controls and safer redaction for WebDAV/cloud credentials, AI keys, and calendar ICS URLs.
- Android builds: ABI‑split arm64 APKs and smaller release output for IzzyOnDroid/F‑Droid.

## Desktop
- Allow file paths in the link attachment prompt.
- Fix diagnostics log clearing.

## Mobile / Android
- New `apps/mobile/scripts/android_build.sh` to build arm64 APKs locally with Gradle.
- ABI splits enabled; size‑reduction flags enabled (minify, resource shrink, legacy packaging).
- GitHub Release workflow updated to use the Gradle build script.

## Docs
- Added IzzyOnDroid recipe draft.
- Updated mobile build instructions.
