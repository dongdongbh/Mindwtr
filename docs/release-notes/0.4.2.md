# Mindwtr v0.4.2

Release date: 2025-12-31

## Highlights
- Multi-filtering for Next and Agenda (contexts/tags, priority, time estimate).
- Task priority metadata across desktop and mobile.
- Project/area filters on the desktop Board and Projects views.

## Desktop
- `e` now opens the selected task editor in list views.
- Board view supports project filters.
- Projects view supports area filtering.
- Blocked-by options are scoped to the current project.
- Native title bar theme follows the app theme; Linux runs borderless with F11 fullscreen.

## Mobile
- Priority field added to task editor and view summary.
- Next and Agenda include lightweight filters for contexts/tags, priority, and time estimate.
